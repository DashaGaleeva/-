5) Контрольный вопрос 8
   Объясните основные этапы алгоритма Кристофидеса для TSP и почему он дает 1.5-аппроксимацию.
   Ответ:
   Основные этапы алгоритма:
   1. Вычисление минимального остовного дерева — набора соединяющих города ветвей без замкнутых петель.
   2. Определение множества вершин с нечётной степенью в минимальном остовном дереве.
   3. Образование подграфа графа, используя лишь вершины множества.
   4. Построение совершенного паросочетания минимального веса в этом подграфе.
   5. Объединение паросочетания и стягивающего дерева для образования эйлерова мультиграфа.
   6. Вычисление эйлерова обхода.
   7. Преобразование цикла, найденного на предыдущем шаге, в гамильтонов цикл путём пропуска повторяющихся вершин (сокращение).
   Аппроксимационный коэффициент 1,5 достигается за счёт комбинации двух решений: минимального остовного дерева и совершенного паросочетания.
