- Биноминальная куча — коллекция биноминальных деревьев, удовлетворяющих свойствам кучи. Каждое дерево имеет степень k, содержит 2ᵏ узлов.
- Куча Фибоначчи — структура с амортизированной эффективностью операций.
- Хеш-таблица — структура данных, реализующая интерфейс ассоциативного массива.
- Особенности реализации:
  - Python:
    - Используют классы для узлов и кучи.
    - Ссылки на дочерние узлы — список или связный список.
    - Консолидация через словарь/список по степеням
    - Простота за счёт динамических типов.
  - Java:
    - Строгая типизация: явные классы и интерфейсы.
    - Использование ArrayList для списка детей и корней.
    - Параметризация и сравнения через Comparable
    - HashMap<K, V> и HashSet<E> — стандартные.
    - Своя реализация: массив LinkedList<Entry<K,V>>[] для цепочек.
  - C++
    - Ручное управление памятью
    - Оптимизированные циклические списки (ручные ссылки).
    - std::unordered_map<K, V> и std::unordered_set<T> — стандартные.
    - Своя реализация: массив векторов (std::vector<std::pair<K, V>>*) или открытая адресация.
    - Шаблоны, кастомные хеш‑функции (std::hash<K>).
