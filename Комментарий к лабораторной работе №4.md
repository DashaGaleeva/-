- Граф — это нелинейная структура данных, состоящая из узлов и ребер. Может быть ориентированным (рёбра имеют направление), неориентированным (рëбра не имеют направление) и смешанным. Может быть взвешанным (рёбра имеют вес) и не взвешанным (рёбра не имеют веса).
- Реализация:
  - На Python удобнее реализовывать через ассоциативные массивы или вложенные списки. 
  - На C++ реализуется через матрицу смежности. 
  - На Java тоже можно реализовывать через матрицу или же использовать существующий класс Graph
- Дерево — это иерархическая структура данных, состоящая из узлов, где каждый узел имеет один родительский узел и может иметь несколько дочерних узлов
- Реализация:
  - На Python реализация идет через списки или словари.
  - На C++ — через указатели.
  - На Java удобнее всего использовать объекты.
