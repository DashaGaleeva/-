- Мультисписок — это структура данных в программировании, каждый элемент которой входит одновременно в несколько односвязных списков.
- Очередь — линейная структура данных, состоящая из последовательности элементов.
- Приоритетная очередь — абстрактный тип данных, который организует элементы на основе их приоритета, а не порядка, в котором они были добавлены.
- Дек — универсальная структура данных, которая представляет собой последовательность элементов, у которой есть два конца. Причём добавление и удаление элементов может происходить как в начало, так и в конец структуры.
- Особенности реализации: На C++, как наиболее типизированном языке, все эти структуры являются встроенными, тогда как на Java и Pyton их необходимо задавать через метод class (в Java - public class)
