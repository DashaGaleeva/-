Вариант 3
1. Задание:
   Задача: реализовать жадный алгоритм для дробного рюкзака. Сортировка предметов по удельной стоимости (value/weight).
   Требования:
   - Считать предметы как объекты с весом и стоимостью
   - Вместимость рюкзака задается как параметр
   - Вывести максимальную стоимость и набор предметов
   Входные данные:
   Предметы: {(вес=2, стоимость=10), (вес=3, стоимость=20),
    (вес=4, стоимость=15), (вес=5, стоимость=25)}
   Вместимость: 8
2. Анализ:
   - Создание списка предметов:
     Каждый предмет представлен объектом класса Item, содержащим три атрибута:
     - weight: вес предмета
     - value: стоимость предмета
     - unit_value: удельная стоимость (стоимость / вес)
   - Сортируем предметы по удельной стоимости (unit_value).
     Чем больше удельная стоимость, тем выгоднее брать этот предмет первым.
   - Итерационный выбор предметов:
     - sorted_items = sorted(items, key=lambda x: x.unit_value, reverse=True): Перебираются предметы в порядке уменьшения удельной стоимости.
     - if item.weight <= remaining_capacity: fraction = 1.0: Если остаток емкости рюкзака позволяет взять предмет целиком, добавляем его полностью.
     - fraction = remaining_capacity / item.weight: Иначе берем столько, сколько вмещается (дробная доля).
   - Выходные данные:
     - print(f"Максимальная стоимость: {max_value:.2f}"): Общая стоимость собранных предметов.
     - print(f"  Вес: {item.weight}, Стоимость: {item.value}, Доля: {frac:.2f}, "
              f"Добавленная стоимость: {entry['added_value']:.2f}, Вес: {entry['added_weight']:.2f}"): Список выбранных предметов с долями каждого.
3. Временная сложность: O(NlogN)
4. Пояснение:
   Основная трудоемкая операция — сортировка предметов по удельной стоимости, реализуемая через цикл с условием, что и приводит к сложности O(NlogN)
