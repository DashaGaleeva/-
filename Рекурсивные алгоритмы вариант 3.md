Задание:
Напишите рекурсивную функцию для нахождения наибольшего общего делителя (НОД) двух чисел, используя алгоритм Евклида.
Анализ:
 Определение функции: 
  Функция называется gcd, принимает два параметра (a и b) и возвращает наибольшее общее число, которое делит оба числа нацело.
 Реализация алгоритма Евклида: 
  Использован рекурсивный подход, согласно которому:
   Если одно из чисел равно нулю, то другое число и есть НОД. if b == 0: return a
   В противном случае алгоритм продолжает свою работу, заменяя пару чисел следующим образом: второе число становится первым, а новое второе число получается путём
   вычисления остатка от деления первого числа на второе. return gcd(b, a % b)
Временная сложность: O(log min(a,b))
Пояснение:
На каждом шаге рекурсии размер чисел уменьшается как минимум в константное число раз (благодаря свойствам модуля).
Контрольный вопрос 8:
Сколько существует перестановок множества из n элементов?
Ответ: n!
